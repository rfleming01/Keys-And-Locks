 # Keys and Locks
**Version: 1.0**

* Starts at a menu screen, which can either quit the game or start the levels
* Game has 4 levels, each with different layouts
* Every level consists of the player character gathering three different colored keys
* Each key opens the correspondingly colored lock
* UI shows what keys you have
* Player character able to move around with arrow keys and space bar
* Moving platform moves between two invisible arrows
* Exit sign takes the player to the next level
* Ladder allows the player to move up it
* Treasure object plays win animation and ends the game


[Go here to play the working game](https://gd.games/games/11d7817f-1b67-47f4-8997-2b0a5ceb5cf4)

* Download the current version of [Gdevelop](https://gdevelop.io/download)
* Download and unzip this source folder
* Open VacuumnMarauders.json in gdevelop
